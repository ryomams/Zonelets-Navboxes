# Zonelets-Navboxes
MediaWiki-style navboxes (nb's) for your vanilla Zonelets site. Originally made for my friend's blog at [https://lakespirit.love/](https://lakespirit.love/), see it live at [https://lakespirit.love/archive](https://lakespirit.love/archive)!
If you don't know what Zonelets is, Zonelets is made by [Marina Kittaka](https://marinakittaka.com), and describes itself as a "simple blogging engine". Learn more at [zonelets.net](https://zonelets.net).

> [!CAUTION]
> This is a somewhat incomplete project (and I'm not on GitHub that much). Use at your own risk! Before installing, please have a backup of the Zonelets site you are modifying in case something goes wrong. Even better, please don't edit your site live! Edit locally, then upload!

## Installation
I'm trying out GitHub releases, so releases of this project can be found in the sidebar of the repository page. Each release is designed around the currently available version of Zonelets, and this will be listed in each release. I don't have a schedule for releases, and I don't plan to make one. This project assumes you are a Zonelets user, and are planning to install this in addition to Zonelets. I haven't tested for compatibility with [other Zonelets mods](https://zonelets.net/legacy), so if you are not using vanilla Zonelets, it's on you to be aware of compatibility.

Here's a couple of steps to help you get started, assuming you are starting with a vanilla setup of Zonelets:
0. Download the latest release from the releases tab. If you're reading this on your machiine, you've already done this :)
1. Read through zlNavboxes.js and follow the steps listed.
2. Do the same for zlNavboxes.html.
3. Do the same for zlNavboxes.css and modify any attributes you desire.
4. I hope it works??????
   - You're doing a lot of testing for me if you choose to install this yourself, so please refer to the [contributing section](#Contributing) if you run into issues.

Documentation and further explanation of this project can be found in the files of the project themselves. I've written each file to be its own guide, so if you need further installation assistance, refer to what's there first.
